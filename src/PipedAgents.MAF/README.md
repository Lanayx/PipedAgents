Wrapper around `Microsoft.Agents.AI` .NET library

### Setup

Set the following environment variables to use the library with **OpenAI**-compatible APIs:
- `OPENAI_API_KEY` - Your API key.
- `OPENAI_BASE_URL` - (Optional) Base URL for API

Set the following environment variables to use the library with **Anthropic**-compatible API:
- `ANTHROPIC_API_KEY` - Your API key.
- `ANTHROPIC_AUTH_TOKEN` - (Optional) Your API auth token.
- `ANTHROPIC_BASE_URL` - (Optional) Base URL for API

Alternatively, you can provide these values directly when creating the client.
